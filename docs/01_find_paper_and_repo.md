# Find Paper And Repo

## 1. 搜索论文

搜索：

```text
Fashion-MNIST: A Novel Image Dataset for Benchmarking Machine Learning Algorithms
```

记录：

- 论文标题
- 作者
- 数据集大小
- 图像尺寸
- 类别数量
- 官方 GitHub 仓库

## 2. 判断官方仓库

官方仓库：

```text
https://github.com/zalandoresearch/fashion-mnist
```

观察证据：

- owner 是 `zalandoresearch`
- README 说明数据集规模、图像尺寸和类别数
- 仓库包含 `benchmark`、`data`、`utils`、`visualization` 等目录
- README 说明很多机器学习库已经内置 Fashion-MNIST API
