# Find Paper And Repo

## 1. 搜索论文

使用Google Scholar

记录：

- bib信息

- 数据集大小
- 图像尺寸
- 类别数量
- 官方 GitHub 仓库

## 2. 在官方仓库中找到 baseline 来源

官方仓库：

```text
https://github.com/zalandoresearch/fashion-mnist
```

进入仓库后找到：

```text
benchmark/convnet.py
```

记录：

- 这个文件位于哪个目录？
- 模型函数叫什么名字？
- 它使用的是 TensorFlow 还是 PyTorch？
- 为什么我们需要把它迁移成 PyTorch 版本？
